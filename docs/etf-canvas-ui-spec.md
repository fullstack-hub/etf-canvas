# ETF Canvas UI/UX 상세 명세서

> `ETF Canvas.pdf` 원본 데이터를 바탕으로 화면 단위의 UI 구성 요소와 사용자 상호작용(Interaction)을 추출한 상세 분석 문서입니다.

## 0. 공통 UI 원칙 (Global Design Principles)
- **Single Page Application (SPA)**: "한 화면에서 종결" – 페이지 전환 없이 한 화면에서 검색, 선택, 시뮬레이션, 피드백이 유기적으로 이루어지는 레이아웃.
- **다크 모드 / 하이라이팅**: 리스트나 캔버스의 특정 요소(모달, 툴팁 등)가 활성화될 때 주변 배경을 어둡게 처리(Dimmed)하여 시각적 집중도 상승 기능 포함.

---

## 1. 검색 영역 (Search Area)
> 좌측 패널(또는 사이드바) 형태로 배치될 가능성이 높은 ETF 탐색기 공간.

### 1.1 검색창 (Search Bar)
- **Input Field**: 일반 텍스트, 숫자(티커), 키워드, 심지어 자연어 질의까지 단일 Input 창에서 유연하게 입력.
- **Auto-complete**: 검색어 입력 시 하단에 연관 ETF 노출.

### 1.2 카테고리 필터 (Category Filters)
- **9가지 기본 분류 버튼**: 칩(Chip) 또는 탭(Tab) 형태로 제공. (국내/해외 대표지수, 섹터/테마, 채권, 원자재, 레버리지/인버스, 혼합, 액티브, New)
- **다중 선택**: "주식형" + "테마형" 등을 중복 클릭(Toggle)하여 교집합 검색 지원.

### 1.3 리스트 영역 (List View)
- **정렬 옵션**: 상단에 [수익률 순] / [AUM 순] 드롭다운 또는 버튼.
- **결과 카드(항목)**: 
  - 기본 상태: ETF 이름, 티커, 간단한 수익률 정도만 표기.
  - **Hover (마우스 오버) 동작**: 특정 ETF 항목에 마우스를 올리면 화면 우측 혹은 툴팁으로 **5대 핵심 정보**(자산, 국가, AUM, 1년 수익률, 운용보수) 패널 노출.
  - **Click (클릭) 동작**: 해당 ETF 선택 시 리스트 하단 영역에 최근 1년 퍼포먼스 차트가 드롭다운/아코디언 방식으로 펼쳐짐.

## 2. 캔버스 영역 (Canvas Area)
> 사용자가 여러 ETF를 시각적으로 모아두고 비교·조합하는 핵심 작업 공간. (중앙 혹은 우측 패널)

### 2.1 캔버스 아이템 (ETF Card)
최대 10개까지 추가되는 캔버스 내부의 개별 ETF 카드 UI 구성:
- **선택 상태 표시**: 포트폴리오 구성을 위해 선택 시 바탕이 '파란색 음영'으로 강조됨. 최대 3개까지 선택 제한.
- **카드 내 컨트롤 아이콘**:
  - `[우측 상단] 체크박스`: 최종 포트폴리오 시뮬레이션 대상에 포함할지 여부 Toggle.
  - `[좌측 상단] 휴지통`: 캔버스 리스트에서 해당 ETF 제거.
  - `[좌측 하단] i (Info)`: 모달 띄우기 (기본 정보, 1년간 %단위 수익률 차트, 상위 10개 보유 종목 리스트). 배경 화면 Dimming 효과 적용.
  - `[우측 하단] Refresh`: **유사 ETF 교체 기능**. 

### 2.2 Refresh 인터랙션 (유사 상품 교체 모달)
1. 카드의 우측 하단 Refresh 버튼이 반짝이는(Blinking) 애니메이션 제공.
2. 버튼 클릭 시, 해당 ETF와 **동일한 기초지수(Benchmark)**를 추종하는 타 운용사의 ETF 목록을 팝업/모달 형태로 나열.
3. 목록에서 다른 ETF 선택 후 "교체" 버튼 클릭.
4. 팝업 소멸, 캔버스 위의 ETF가 새 상품으로 UI 트랜지션 되며 변경.

## 3. 결과/저장 및 피드백 영역 (Result & Feedback)
> 캔버스에서 선택된(최대 3개) ETF를 바탕으로 시뮬레이션하고 결과를 보여주는 영역. 화면 하단 혹은 우측 하단.

### 3.1 비중 설정 (Weight Setup)
- **Input / Slider**: 선택된 ETF 별로 슬라이더나 텍스트 박스를 활용해 총합이 100%가 되도록 비중 설정.
- **자동 반영**: 캔버스에서 ETF를 클릭(상태 변경)할 때마다 이 비중 설정 리스트에 즉각 추가 혹은 제거됨.

### 3.2 합성 패널 (Simulation Board)
- **[합성] 버튼**: 클릭 시 (Login 필요시 소셜 로그인 모달 호출) 차례 진행.
- **결과 차트**: 좌측에 포트폴리오 합성 수익률 차트 렌더링. 
  - **확대 기능(Zoom)** 버튼 제공하여 차트를 화면 전체 혹은 크게 볼 수 있게 지원.
- **결과 수치**: 하단에 종합 1년 수익률, 종합 MDD 수치 텍스트 표기.

### 3.3 피드백 및 유도 (Feedback Banner)
- **AI/시스템 조언 문구**: 합성 버튼 클릭 시 포트폴리오의 편향성이나 개선점을 나타내는 텍스트 배너 노출.
- **액션 버튼**: 배너 마지막에 조언에 따른 액션 유도 버튼 배치 (예: `[채권형 ETF 조회하기]`).
- **상호 작용**: 버튼 클릭 시, 1번(검색 부분) UI로 상태가 리셋/업데이트 되며 필터링 된 새 검색 결과를 렌더링 (순환 구조).

### 3.4 내 포트폴리오 (My Portfolios - Modal/Drawer)
- 유저 마이페이지 공간.
- **리스트 컨테이너**: 수직 스크롤 바운스.
- **항목 표기**: 이름, 날짜, 태그(포함 ETF), 1년 수익률, MDD.
- **필터 제어**: 최신순 / 수익률순 / MDD순 탭 컨트롤.
